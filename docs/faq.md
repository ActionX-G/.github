# FAQ

## What is ActionX?

ActionX is a commerce infrastructure solution for AI products.
It helps identify buying intent in conversations and return monetizable recommendation payloads via API, SDK, or MCP integration.

## Is ActionX only for large products?

No. ActionX is built to support monetization from day one, including early-stage products with low or moderate traffic.

## Which product types are a fit?

ActionX can fit:

- AI chat applications
- AI agents and agent platforms
- AI search and recommendation products
- workflow tools with conversational interfaces

## Do we need to change our front-end design?

Not necessarily. ActionX returns raw, structured payloads so your team can keep control over UI presentation and placement.

## Does ActionX support MCP-based ecosystems?

Yes. ActionX supports integration paths that include MCP-friendly workflows, in addition to API and SDK options.

## How is monetization measured?

Depending on setup, teams can monitor click and conversion-related outcomes and optimize strategy over time.
