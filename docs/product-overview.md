# Product Overview

ActionX is designed for teams that want to monetize AI traffic without waiting for massive scale.

## Core Capabilities

### 1) Intent Detection in Conversation

ActionX analyzes conversational context to identify buying intent in AI chat or agent workflows.

### 2) Product and Offer Matching

When intent is detected, ActionX matches users with relevant:

- products and brands
- cashback offers
- affiliate opportunities
- sponsored recommendations

### 3) Flexible Delivery via API / SDK / MCP

ActionX supports integration methods that fit modern AI stacks:

- API for backend orchestration
- SDK for application-level embedding
- MCP for tool-based and agentic ecosystems

### 4) Raw Payload Control

Instead of forcing one display style, ActionX returns structured payloads so teams can render content in the way that best fits their UX:

- cards
- inline suggestions
- recommendation blocks
- contextual links

### 5) Operational Visibility

ActionX provides measurement and controls around:

- clicks
- conversions
- commission outcomes
- content strategy tuning

## Why Builders Choose ActionX

- monetize from day one, even with low traffic
- align monetization with AI-native user behavior
- keep full control over product experience
- support agentic commerce and AI search monetization use cases

## Typical Integration Flow

```text
User asks AI
  -> intent recognized
  -> commerce candidates selected
  -> recommendation payload returned
  -> product UI decides how to render
  -> click / conversion tracked
```
